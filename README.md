## Install:
```bash
> pkg update && pkg upgrade -y
> pkg install git
> pkg install ffmpeg
> pkg install nodejs
> git clone https://github.com/syahdanprojectID/youtubedl
> cd yotubedl
> npm install
```

### Run
```bash
> node syahdan.js
```
